Guia de Contribuição O Guia de Contribuição e Estilo serve para orientar a forma de trabalhar que seja eficiente, padronizada, coordenada e segura e produza conteúdo de qualidade. Este guia contém orientações de como escrever o código e escrever a documentação. Conjunto de Dados Para um melhor aproveitamento dos projetos coletivos do PET é recomendado o uso do LabestData quando for necessário o uso de um conjunto de dados.
